The zero-data screen — what users see before they've created anything, or when a search or filter returns nothing. Choose this to turn a dead end into an invitation. A well-designed empty state reduces confusion and accelerates the first meaningful user action. The illustration or icon sets the tone; the headline and CTA complete the message. Pairs with any aesthetic; keep the illustration style consistent with the product's visual language.
