A grid of asymmetric, self-contained card cells arranged in a dense mosaic. Each cell is independent — different sizes, different content types — creating visual rhythm through contrast of scale. Cells span multiple columns or rows for hierarchy. No explicit dividers; the gaps between cells define structure. Inspired by Japanese bento organization: everything has a place, nothing is wasted.
