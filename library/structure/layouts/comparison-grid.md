Purpose-built for structured comparison — product specifications, feature matrices, competitive analysis, or technical option comparison. Choose this when users need to evaluate multiple items across the same set of attributes. Unlike a pricing table (which is conversion-oriented), this layout is research-oriented — dense, thorough, and scannable. Pairs with a clean, functional aesthetic and a compact typography spec. Works well within a product-detail or pricing-table page as a supplementary section.
