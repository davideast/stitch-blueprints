Built for technical documentation, API references, and knowledge bases. Choose this when content is organized into many interconnected pages users navigate non-linearly. The three-column structure keeps context (left nav) and position (right nav) always visible without disrupting reading flow. Pairs with a monospace or clean sans-serif typography spec and a subtle tonal color scheme for code blocks.
