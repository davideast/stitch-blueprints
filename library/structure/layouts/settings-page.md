The standard configuration and preferences layout. Choose this when users need to configure multiple independent settings grouped by category. The label-left, control-right pattern creates a scannable list that users can skim for the setting they need. Sections prevent overwhelming users with a wall of options. Pairs with a clean, functional aesthetic and a compact typography spec. Works for both user preferences and admin system configuration.
