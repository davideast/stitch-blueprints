A monospaced, syntax-highlighted block for displaying code snippets. Has a header bar showing the language name and a copy-to-clipboard button. Line numbers are optional. The background is a slightly elevated surface color — distinct from the page background but not jarring. Long lines scroll horizontally rather than wrapping. Supports multiple tabs for multi-file examples.
