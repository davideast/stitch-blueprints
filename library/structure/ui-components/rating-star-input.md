A row of 5 stars (or other icons) that accepts a user rating via click or tap. Filled stars use the tertiary/accent color; empty stars use outline style in a muted tone. Hovering previews the selection before committing. Optionally accompanied by a label showing the current rating value ("4 out of 5") and a count of total ratings. Can be read-only for display or interactive for input.
