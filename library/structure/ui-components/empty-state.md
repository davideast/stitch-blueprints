slot: ui-component
type: structural
tags: [zero-data, onboarding, blank-slate, first-use]
---

A centered composition shown when a list, feed, or collection has no items. Includes an illustration or icon, a brief headline explaining the empty condition, a supporting sentence with context or encouragement, and a single primary CTA to create the first item or take the first action. The tone matches the product voice — never clinical, never apologetic.
