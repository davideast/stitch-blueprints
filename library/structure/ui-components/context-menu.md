  Show a floating list of actions anchored to a trigger (right-click, long-press, or
  overflow ⋯ button). Each item has a single action label with optional leading icon.
  Destructive actions appear at the bottom, separated by a divider, styled in error color.
  Dismiss on outside click or any selection.

A floating action menu triggered by right-click, long-press, or overflow button. Choose this when you need contextual actions on items — file operations, row actions, or any multi-action trigger point. Destructive actions are visually separated and error-colored. Pairs well with any layout that has actionable list items or cards.
