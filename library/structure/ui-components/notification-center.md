A slide-in panel or dropdown tray listing recent notifications grouped by recency (Today, Yesterday, Earlier). Each notification has a leading icon or avatar, a title, a brief description, and a timestamp. Unread items are visually distinguished — a colored dot or subtly elevated background. Includes a "Mark all read" action and a link to full notification history.
