A dense chronological event log — tighter and more utilitarian than a card feed. Choose this for audit trails, project activity sidebars, social notifications, or system logs where recency and scannability matter most. Groups by day for long lists. Pairs well with infinite-scroll-feed layouts and notification-center components.
