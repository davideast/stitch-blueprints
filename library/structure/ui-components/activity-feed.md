A chronological list of discrete events, each represented by an avatar or icon, a description of the action, and a relative timestamp. Items are tightly spaced — this is a dense log, not a card list. Recent items appear at the top. Grouped by day when the list is long. Used for audit trails, social notifications, project activity, and system logs.
