Labels are uppercase with wide letter-spacing — the classic UI chrome convention. Small size (11–12px), medium weight. Used for section titles, button text, metadata tags, and navigation items. The uppercase treatment gives the interface a structured, publication-like quality and clearly delineates UI chrome from content.
