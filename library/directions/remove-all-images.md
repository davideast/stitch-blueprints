## When to use
When imagery is weak, inconsistent, or distracting from the message. Use this radical reduction to test whether the layout and typography can carry the design without photographic support.

## What it moves
Removes all img elements and background-image properties. Sections that relied on photography must restructure around type scale, color blocks, and spacing. The layout shifts from image-driven to purely typographic.
