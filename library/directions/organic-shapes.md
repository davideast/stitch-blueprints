## When to use
When every element is a rectangle and the layout feels mechanical. Use this to inject personality and natural warmth by breaking geometric rigidity.

## What it moves
Applies clip-path polygons and ellipses, SVG blob shapes, and irregular border-radius values to containers and decorative elements. Straight edges become curved. Grid-locked layouts gain flowing, asymmetric forms that feel hand-drawn rather than machine-cut.
