"Collapse to a list. No cards, no images. Just a clean typographic list."

## When to use
Design uses cards or grid components when the underlying content is actually a list. The containers are adding visual noise without adding meaning.

## What it moves
Removes all card containers and imagery, presents content as a clean typographic list — title, supporting label, and a rule or divider. References editorial indexes, press release lists, minimalist app tables.
