## When to use
When background imagery competes with foreground content for attention. Use this to create a clear focal plane that separates content from context.

## What it moves
Applies CSS filter blur to background images and decorative layers. Foreground content stays crisp while backgrounds soften. Creates a depth-of-field effect that sharpens the visual hierarchy between content and environment.
