"Double the whitespace. Every gap, every margin — double it."

## When to use
Design feels cramped or busy when it should feel considered. Content is close together in ways that read as default rather than deliberate.

## What it moves
Multiplies all spacing values — margins, padding, gaps between elements — creating a dramatically more airy composition. Content begins to feel placed rather than stacked. Shifts toward luxury editorial rhythm.
