## When to use
Navigation is a small utility bar when it could be the primary editorial statement. The links themselves are the content.

## What it moves
Promotes navigation links to display type scale, treats the nav as a typographic layout — each link a headline in its own right. References editorial magazine section pages, portfolio index pages, typographic site maps.
