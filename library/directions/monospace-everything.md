"Monospace everything. Every label, every number, every heading."

## When to use
Design should feel like a precision instrument or technical interface but has humanist or proportional type that softens it.

## What it moves
Replaces all typefaces with monospace, introduces terminal/instrument aesthetic, aligns numbers naturally by column. References Bloomberg Terminal, code editors, engineering dashboards.
