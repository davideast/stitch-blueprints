## When to use
When sections are cluttered with competing elements and nothing has clear priority. Use this to force every section to commit to one focal point.

## What it moves
Removes secondary content, supporting text, and supplementary visuals from each section. Only one dominant element remains per viewport block. Density collapses. Each section becomes a single statement surrounded by space.
