"Repeat the element. The pattern is the design."

## When to use
Design has a single interesting component that appears once when its repetition would create rhythm and visual system.

## What it moves
Takes the most visually interesting element and repeats it — same component, slight variation each time (label, number, state). The serial arrangement becomes the layout. References data tables, leaderboards, activity feeds, product grids.
