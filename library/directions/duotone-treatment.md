## When to use
When photography feels inconsistent — different lighting, color temperatures, styles. Duotone forces visual cohesion across all imagery by flattening to a shared palette.

## What it moves
Applies CSS filter combinations (grayscale + sepia + hue-rotate) or blend modes to map images to two palette colors. Removes original color information from photos. All imagery shifts to a unified tonal range regardless of source.
