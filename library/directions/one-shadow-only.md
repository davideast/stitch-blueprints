"Flatten everything except one element. Give the shadow to the thing that matters."

## When to use
Design uses shadow liberally to create depth across many elements. Elevation is everywhere, so it means nothing.

## What it moves
Removes shadow from everything except the single most important element — the primary CTA, the key metric, the focal card. That one shadow now carries enormous visual weight because it's alone.
